= Flujo de trabajo de instalación de software para interruptores BES-53248
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para instalar y configurar el software para un conmutador de clúster BES-53248, siga estos pasos:

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1"]link:configure-install-initial.html["Configura el interruptor"]
[role="quick-margin-para"]
Configurar el conmutador de clúster BES-53248.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dos"]link:configure-efos-software.html["Instale el software EFOS"]
[role="quick-margin-para"]
Descargue e instale el software Ethernet Fabric OS (EFOS) en el conmutador de clúster BES-53248.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Tres"]link:configure-licenses.html["Instale las licencias para los conmutadores de clúster BES-53248."]
[role="quick-margin-para"]
Opcionalmente, agregue nuevos puertos comprando e instalando más licencias.  El modelo base del switch tiene licencia para 16 puertos 10GbE o 25GbE y dos puertos 100GbE.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["Cuatro"]link:configure-install-rcf.html["Instalar el archivo de configuración de referencia (RCF)"]
[role="quick-margin-para"]
Instale o actualice el RCF en el conmutador de clúster BES-53248 y, a continuación, verifique los puertos para una licencia adicional después de que se haya aplicado el RCF.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-5.png["Cinco"]link:configure-ssh.html["Habilitar SSH en los conmutadores del clúster BES-53248"]
[role="quick-margin-para"]
Si utiliza las funciones de monitorización del estado del conmutador Ethernet (CSHM) y de recopilación de registros, habilite SSH en los conmutadores.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-6.png["Seis"]link:reset-switch-bes-53248.html["Restablecer el interruptor a los valores predeterminados de fábrica"]
[role="quick-margin-para"]
Borre la configuración del interruptor del grupo BES-53248.
