= Flujo de trabajo de actualización para conmutadores de clúster BES-53248
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Siga estos pasos para actualizar el software EFOS y los archivos de configuración de referencia (RCF) en los conmutadores de clúster Broadcom BES-54328, según corresponda.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["1"]link:upgrade-efos-software.html["Actualiza tu versión de EFOS"]
[role="quick-margin-para"]
Descargue e instale el software Ethernet Fabric OS (EFOS) en el conmutador de clúster BES-53248.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["Dos"]link:upgrade-rcf.html["Actualiza tu versión de RCF"]
[role="quick-margin-para"]
Actualice el RCF en el conmutador de clúster BES-53248 y, a continuación, verifique los puertos para una licencia adicional después de que se aplique el RCF.

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["Tres"]link:replace-verify.html["Verifique la red del clúster ONTAP después de la actualización"]
[role="quick-margin-para"]
Verifique el estado de la red del clúster ONTAP después de una actualización del software EFOS o RCF para los conmutadores de clúster BES-53248.
