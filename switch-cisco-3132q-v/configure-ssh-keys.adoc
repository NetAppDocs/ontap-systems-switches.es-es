= Verifique su configuración SSH
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si está utilizando las funciones de monitorización del estado del conmutador Ethernet (CSHM) y recopilación de registros, verifique que SSH y las claves SSH estén habilitadas en los conmutadores del clúster.

.Pasos
. Verifique que SSH esté habilitado:
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. Verifique que las claves SSH estén habilitadas:
+
`show ssh key`

+

Ver https://docs.netapp.com/us-en/ontap/networking/configure_network_security_using_federal_information_processing_standards_@fips@.html#enable-fips["Configure la seguridad de la red utilizando FIPS."^] Para obtener más detalles.
