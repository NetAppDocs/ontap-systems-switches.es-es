= Compruebe su configuración SSH
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Si está utilizando el Monitor de estado del conmutador Ethernet (CSHM) y las funciones de recopilación de registros, verifique que SSH y las claves SSH estén habilitadas en los conmutadores.

.Pasos
. Compruebe que SSH está habilitado:
+
[listing, subs="+quotes"]
----
(switch) *show ssh server*
ssh version 2 is enabled
----
. Compruebe que las claves SSH estén habilitadas:
+
`show ssh key`

+
