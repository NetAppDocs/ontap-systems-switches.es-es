= Guía de inicio rápido para sistemas AFX, clúster, almacenamiento y switches compartidos
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Para comenzar a trabajar con sistemas AFX, clústeres, almacenamiento y conmutadores compartidos, debe instalar componentes de hardware y configurar su conmutador.

El despliegue del switch implica el siguiente flujo de trabajo.

[role="quick-margin-para"]
